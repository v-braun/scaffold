### CocoaPods
<%= name %> is available on [CocoaPods](http://cocoapods.org). Just add the following to your project Podfile:

```ruby
pod '<%= name %>'
```

Or from GitHub:

```ruby
pod '<%= name %>', :git => 'https://github.com/<%= author %>/<%= name %>.git'
```

### Manually
1. Download and drop ```file-name.swift``` in your project.  
2. Congratulations!
